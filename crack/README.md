# 🏗️ Surface Crack Detection AI

An AI-powered web app that detects structural cracks in surface images using a trained CNN model.

## 🚀 Live Demo
[Click here to try the app](https://share.streamlit.io)

## 🧠 How It Works
- Upload a surface image or use your camera
- The CNN model analyzes the image
- Get instant results with confidence percentage

## 🛠️ Tech Stack
- **Python**
- **TensorFlow / Keras** — CNN model
- **Streamlit** — Web interface
- **Pillow** — Image processing

## 📦 Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
